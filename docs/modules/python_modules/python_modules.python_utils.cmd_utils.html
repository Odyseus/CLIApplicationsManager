Parameters

  • cmd (list) – The command to run.

  • stdout (None, optional) – Pipe to be connected to the standard output stream.

  • stderr (None, optional) – Pipe to be connected to the standard error stream.

  • output_stream (int, optional) – Which output streams should be used (stdout, stderr or both).

  • env (None, optional) – A mapping object representing the string environment.

  • cwd (None, optional) – Path to working directory.

  • logger (object) – See <class LogSystem>.
