bootstrap_css_static()[source]¶

Serve the file found at UserData/www/assets/css/bootstrap.min.css.

This allows me to use the Bootstrap CSS stylesheet from any HTML document located at any directory depth. I can simply use <link rel="stylesheet" href="/_assets_bootstrap_css" /> and the desired stylesheet will be used.
