handle_all_repositories()[source]¶

Handle all repositories.

The main tasks of this method are to populate the self._data_tables_obj list with all repositories data and to copy desired files into the www folder. The self._data_tables_obj is finally saved into a JSON file for easy retrieval.

Repositories whose handler is sphinx_docs will have their DataTables data generated, but are purposely handled (the Sphinx documentations built) separately in self.build_sphinx_docs.
