
******************************
Chroot Helpers's documentation
******************************

A :abbr:`CLI (Command Line Interface)` utility written in Python 3 to assist in the basic setup of a chroot environment to be able to perform rescue operations.

Requirements
============

.. attention::

    No mayor requirements are needed to run this application other than Python 3 (Python 3.5+ to be precise).
